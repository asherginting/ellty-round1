## ⌛️ Installation

1. Clone this repo
```
git@github.com:asherginting/ellty-round1.git
```

2. cd folder
```
ellty-round1
```
3. Install module
```
pnpm install
```

4. Done, ```pnpm run dev``` to run the project

<br>

Runs the app default in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
